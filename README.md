# Portfolio Website

This is the portfolio of Aman Singh.

## Getting Started

Welcome to my Angular portfolio! It features:
- 3D interactive models using Three.js
- Theme switching capability (3 different looks)
- Modern Angular-based architecture

### Prerequisites

You need to have npm and Angular CLI installed:
```bash
npm install -g @angular/cli
```

### Installing

```bash
npm install --force
```

### Running the project

```bash
npm start
```

Go to [http://localhost:4200](http://localhost:4200) after compile finishes.
